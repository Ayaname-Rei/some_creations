# RoboGame 搭车日志

这是 RoboGame 团队生活记录网页的第一版静态设计稿。

## 文件

- `index.html`：页面结构与占位文字
- `styles.css`：视觉样式与响应式布局
- `script.js`：滚动出现动画与导航状态

## 后续替换建议

1. 在仓库中新建 `assets/photos/`，放入工作照、搭车照、合影和赛场照。
2. 在相册区把 `.placeholder` 替换为 `<img src="assets/photos/xxx.jpg" alt="...">`。
3. 在视频区把 `.video-slot` 替换为 `<video>` 或 iframe 嵌入。
4. 将队员卡片中的姓名、分工和占位描述替换为真实内容。

直接用浏览器打开 `index.html` 即可预览。
